# DS-ML
Welcome to my repository! It contains my independent projects in Data Science and Machine Learning engineering.
The projects cover practical data-driven applications such as predictive modelling, reinforcement learning and some software development.

Each project is contained in its own folder, with a **README** for a quick introduction, a **docs/** folder with detailed Markdown documentation, and a **src/** folder with the project code.

I recommend that you start by exploring the **Project Search** app, which is a search-based recommender that allows you to find projects based on skills or frameworks. You can access the app here: https://portfolio-search.streamlit.app/
